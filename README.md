SplitPay
Description
SplitPay is a hackathon project designed to simplify and manage shared expenses by splitting payments among multiple users in an efficient and transparent way. The application helps users calculate, track, and manage shared payments easily.
________________________________________
Installation Guide
Follow these steps to set up SplitPay locally:
1.	Clone the repository
git clone https://github.com/your-username/splitpay.git
2.	Navigate to the project directory
cd splitpay
3.	Install dependencies
pip install -r requirements.txt
4.	Run the Flask application
python app.py
5.	Open your browser and visit
http://127.0.0.1:5000
________________________________________
Usage
Once SplitPay is running on your local server:
1.	Open your browser and go to: http://127.0.0.1:5000
2.	Enter the required expense details:
o	Total expense amount

o	Number of people involved

o	Any additional details (if applicable)
3.	Click Split / Calculate
4.	View the results:
o	Split amount per person

o	Clear and accurate calculation based on inputs
Example Use Case:
A group of friends goes out for a meal. One person pays the total bill. Enter the total amount and number of people into SplitPay, and the app calculates how much each person owes.
Intended Users:
- Friends sharing expenses
- Roommates managing monthly costs
- Small groups splitting bills during trips or events
Notes:
- Runs locally; no internet connection needed after setup
- Data is not stored permanently (no database in this version)
- Designed for learning, hackathons, and demonstrations
